Blinklytics is an interactive Streamlit dashboard built to analyze Blinkit grocery sales data.

## Features
- KPI cards (Total Sales, Avg Sales, Items, Rating)
- Fat Content analysis
- Item type & outlet insights
- Interactive filters

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## How to Run
pip install -r requirements.txt  
streamlit run app.py
